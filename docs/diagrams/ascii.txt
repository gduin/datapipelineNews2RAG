          DATA FLOW (per article)
┌──────────────────────────────────────────────────────────────┐
│  RSS/HTTP fetch  →  Kafka (news.raw)  →  Flink ProcessFuncion │
│                                      ┌──────────────────────┐│
│                                      │ open():              ││
│                                      │  ┌─ configure_logging││
│                                      │  ├─ NormalizeStep()  ││
│                                      │  ├─ SentenceChunker()││
│                                      │  ├─ build_embedder() ││
│                                      │  ├─ QdrantSink()     ││
│                                      │  ├─ ensure_collection ││
│                                      │  └─ DOWN (lazy init) ││
│                                      │                       ││
│                                      │ map() per message:    ││
│                                      │  1. decode (JSON)     ││
│                                      │  2. validate (Pydantic││
│                                      │  3. normalize          ││
│                                      │  4. chunk (512 tok,   ││
│                                      │     overlap 64)       ││
│                                      │  5. embed (768-dim)   ││
│                                      │  6. upsert → Qdrant   ││
│                                      │  7. return {"url",    ││
│                                      │     "chunks": N}     ││
│                                      └──────────────────────┘│
│       → Qdrant (cosine similarity, deterministic UUID ids)   │
└──────────────────────────────────────────────────────────────┘

          QUERY FLOW (/ask endpoint)
┌────────────────────────────────────────────────────────────┐
│  User question                                             │
│         │                                                  │
│         ▼                                                  │
│  embed(query) → Qdrant query_points (top_k=5, cosine)     │
│         │                                                  │
│         ▼                                                  │
│  RetrievedDoc[] (url, title, chunk_text, score)            │
│         │                                                  │
│         ▼                                                  │
│  build_context(docs) → formatted context block             │
│         │                                                  │
│         ▼                                                  │
│  LLMClient.complete(system_prompt, context + question)     │
│         │                                                  │
│         ├── llamacpp → llama-server (OpenAI-compat, v1)    │
│         ├── openai   → OpenAI API                          │
│         └── stub     → returns context verbatim (offline)  │
│         │                                                  │
│         ▼                                                  │
│  RagAnswer(answer, sources=[url, ...])                     │
│         │                                                  │
│         ▼                                                  │
│  HTTP 200 JSON: {"answer": "...", "sources": [...]}        │
